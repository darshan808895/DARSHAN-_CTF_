# WestWild CTF Machine

## Description

 Linux-based Capture The Flag (CTF) machine created for cybersecurity practice. It includes web enumeration, hidden clues, steganography, SSH access, and privilege escalation challenges.

## Services

* SSH (Port 22)
* HTTP (Port 80)

## User Accounts

* a1
* a2
* a3

User credentials are hidden within the machine and must be discovered by enumeration.

## Challenges

* Hidden pages and directories
* robots.txt enumeration
* HTML comments
* JavaScript and CSS clues
* Base64 encoded strings
* Steganography using Steghide
* User and root flags

## Flags

User Flag:

```text
/home/a1/user.txt
```

Root Flag:

```text
/root/root.txt
```

## Tools Used

* Nmap
* Curl
* SSH
* Base64
* Steghide
